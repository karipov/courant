# Courant 
Courant is a bot that lets you access your favorite resources for news and other media in a single chat.

## Resource modes
- Telegram Channels
- RSS Feeds

## Standard features
- Filter keywords
- Summary and agglamation via telegra.ph
- Resources and News recommendations

## Premium Features
- Lower fetch times of RSS and Telegram Information

## Development notes
The use of a NoSQL database permits the data strcutures to be variable, so new 
functionality can be added quickly. However, it must be ensured that everthing
is backwards compatible.
